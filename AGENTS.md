# Agent operating rules

1. Read `skills/time-travel-history-producer/SKILL.md` and the references it routes to.
2. Read `knowledge/LEARNING_REPORT.md` before starting a new episode or revising the
   workflow.
3. Preserve the sequence: sources → raw story → edited storyline → script → continuity
   → shots → QA → packaging → performance.
4. Never commit raw video, generated media, credentials, personal data or masters.
5. Record reusable user feedback, QA failures, reference patterns and metric signals
   before ending the relevant work session.
6. Promotion requires evidence, a canonical rule/template update and validation.
7. Do not silently overwrite or delete learning history; supersede or retire it.
8. Run `python3 skills/time-travel-history-producer/scripts/validate_repo.py` before
   committing.
