## Retention and clarity review

The full board contains five complete escalation loops rather than a flat chronology:

1. **Make the group ordinary → ledger may expose it → gate records the lie.**
2. **Keys repeatedly work → final key fails → quiet plan becomes forced entry.**
3. **Four look powerful → they must hide → recognizable prisoner crosses alone.**
4. **Disguise earns salutes → barrier remains down → Piechowski must perform authority.**
5. **Car creates freedom → identity description spreads → car becomes a liability.**

The late report reveal creates a second mission rather than functioning as an appended
fact card.

## Visual-relevance exceptions

Panel 802 is conceptually correct after correction—the host looks back toward roll
call—but generated empty gaps remain less legible than the paper plan. In production,
this should be deliberately blocked or composited as five unmistakable vacant places,
not left to a generative wide crowd. This is a shot-execution note, not a reason to
redesign the storyline.

No panel uses an arbitrary object metaphor. The mechanism objects in 101–103 and 908
are script-native and physically introduced by the host.

   as host plus four men.

   as five visible people. It was regenerated toward a visibly incomplete roll-call
   formation and flagged for deterministic production blocking.

2. The four men need dedicated silhouette/face reference sheets before production
   generation; current boards prove roles and height differences, not final likeness.
3. Steyr geometry, plain uniform layers, HWL layout and outer-checkpoint geography need
   specialist reference boards before color footage.
4. Panel 802's five empty gaps should use deterministic compositing or a controlled
